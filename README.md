# CraigslistScraper
Craigslist Scraper is a web scraper for craigslist, Users define their searches then Craigslist Scraper pulls all ads and ad data from the defined search and places it neatly inside of a JSON file. The JSON file might look something like: 
<p>
  <img src="img/json_file_example.png" alt="Json Example" width="95%">
</p>
Only tested on macOS with python3.
